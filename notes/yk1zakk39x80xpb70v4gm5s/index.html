Network value and integrated value

  • (Definition.) Integrated value. Define the integrated value analogously to the portfolio value of a firm; recall that the portfolio value of firm iii, pip_ipi​, is defined pi=∑j=1nWijvjp_i = \sum_{j=1}^{n}W_{ij}v_jpi​=∑j=1n​Wij​vj​, the sum of value of each other firm jjj in the group multiplied by the percentage ownership of firm iii in that firm jjj.

  • Integrated value is simply the integrated ownership analogue to portfolio value: νi\nu_iνi​, the integrated value of firm iii, is defined νi=∑j=1nW^ijvj\nu_i = \sum_{j=1}^{n}\hat{W}_{ij}v_jνi​=∑j=1n​W^ij​vj​, based on the integrated ownership matrix rather than the direct ownership matrix

  • In matrix form, we have νint=W^v=(I−W)−1Wv\nu_{int}=\hat{W}v=(I-W)^{-1}Wvνint​=W^v=(I−W)−1Wv.

    • The integrated value is a Level 3 network measure: it fully considers both the connections between nodes (firm ownership) and the nodes themselves (the value of the firms)

    • (Proof.) From the above equation, (I−W)νint=Wv(I-W)\nu_{int}=Wv(I−W)νint​=Wv and thus νint−Wνint=Wv\nu_{int}-W\nu_{int}=Wvνint​−Wνint​=Wv, νint=Wνint+Wv\nu_{int}=W\nu_{int}+Wvνint​=Wνint​+Wv.
    • This gives rise to the following scalar equation:
      νkint=∑i=1nWkiνiint+∑i=1nWkivi\nu^{int}_k=\sum_{i=1}^n W_{ki}\nu^{int}_{i} + \sum_{i=1}^n W_{ki}v_{i}νkint​=i=1∑n​Wki​νiint​+i=1∑n​Wki​vi​
    • The second term ∑i=1nWkivi\sum_{i=1}^n W_{ki}v_{i}∑i=1n​Wki​vi​ is simply the portfolio value of firm kkk.
    • (Interpretation 1.) The integrated value of firm kkk is equal to the sum of the integrated value of other firms multiplied by the ownership of firm kkk in these firms, plus the portfolio value of firm kkk.
    • (Interpretation 2.) The importance of node kkk in the graph depends on the importance of its surrounding nodes and the strength of the connections between kkk and these nodes, as well as the value of these nodes themselves.

  • (Definition.) Network value. Network value is analogous to group value defined previously:

    vnet=Wvnet+vv^{net} = Wv^{net} + vvnet=Wvnet+v

    and hence

    vnet=(1−W)−1vv^{net}=(1-W)^{-1}vvnet=(1−W)−1v

  • (Interpretation.) We can treat a firm's network value as the portfolio of its neighbors' network values, plus its own intrinsic value.

  • Network value satisfies Wvnet=W^v=νintWv^{net}=\hat{W}v=\nu^{int}Wvnet=W^v=νint as shown in the previous sections; as such vnet=Wvnet+v=W^v+v=νint+vv^{net}=Wv^{net}+v=\hat{W}v+v=\nu^{int}+vvnet=Wvnet+v=W^v+v=νint+v.

  • (Theorem.) The network value and integrated values are related to one another by

    vnet=νint+vv^{net} = \nu^{int}+vvnet=νint+v

  • (Interpretation.) The network value of a firm is its underlying value vvv plus the value gained from its integrated value νint\nu^{int}νint, which in turn is equal to the value gained from the underlying value of all firms reachable through indirect ownership.
