Self-Loop Problem and Refinements

Self-loop problem and refinements for group ownership

  • (Problem 1). Self-loops.

    • If firm iii owns shares in firm jjj and firm jjj also owns shares in firm iii, a self-ownership loop is created: iii owns jjj owns iii owns jjj owns iii...

    • This creates a magnifying effect on the final group value because group value is defined recursively: vG=v+WvGv^G=v+Wv^GvG=v+WvG, and if a firm owns a large part of itself indirectly then its own group value will be added onto itself

    • vGv^GvG will often be much larger than ∑vi\sum v_i∑vi​, the total inherent (market) value of the firms

      • Interpretation: self-loops overestimate the group values of strongly connected components (SCCs).

    • This can be refined by looking at the scalar equation for the integrated ownership matrix:

      Wij^=Wij+∑kW^ikWkj\hat{W_{ij}}=W_{ij}+\sum_{k}\hat{W}_{ik}{W}_{kj}Wij​^​=Wij​+k∑​W^ik​Wkj​

      which leads to self-loops due to the term W^iiWij\hat{W}_{ii}W_{ij}W^ii​Wij​, where W^ii\hat{W}_{ii}W^ii​ indicates a firm's indirect ownership of itself. We simply remove this term from the equation:

    • (Refinement 1.) Simply write

      Wij~=Wij+∑kW^ikWkj−W^iiWij\tilde{W_{ij}}=W_{ij}+\sum_{k}\hat{W}_{ik}{W}_{kj} -\hat{W}_{ii}W_{ij}Wij​~​=Wij​+k∑​W^ik​Wkj​−W^ii​Wij​

      removing the self-looped term.

      as the matrix (I−diag(W^))(I-\text{diag}(\hat{W}))(I−diag(W^)) is purely diagonal and has an inverse equal to the inverse of its entries.

    • If W^kk=0\hat{W}_{kk}=0W^kk​=0, i.e. a firm does not own any shares in itself (some economists interpret this as owning treasury shares), vkGv_k^GvkG​ is simply vkv_kvk​, the value of the firm, plus the value of other firms in the group weighted by their integrated ownership by firm kkk.

    • If W^kk>0\hat{W}_{kk}> 0W^kk​>0, the group value is larger than simply the integrated ownership-weighted sum.

  • (Drawbacks.)

    • The issue of self-loops are indeed correctly isolated by Baldone et al., but their modifications to the equation are mysterious (diag(W^)\text{diag}(\hat{W})diag(W^) in particular).
    • The above equation does not actually result in smaller group values if self-loops are present, as if W^kk>0\hat{W}_{kk}>0W^kk​>0 then vkGv_k^GvkG​ still gets very large.
    • The next section will propose a simpler modification to the model that bypasses the self-loop problem.
